# Personal Portfolio

This is a simple, single-page personal portfolio website built as per the **Portfolio Creation** guidelines. It is plain HTML, CSS, and a tiny bit of JavaScript, so it can be hosted easily on any static hosting platform (GitHub Pages, Netlify, Vercel, etc.).

## Structure

- `index.html` – main page with all sections:
  - About / Hero
  - Resume
  - Education
  - Skills
  - Projects
  - Certificates
  - Achievements
  - Internships & Work Experience
  - Competitive Programming & Hackathons
  - Research & Publications
  - Contact
- `styles.css` – modern responsive styling (no framework).
- `script.js` – small enhancement for the current year in the footer.

## How to run

Just open `index.html` in your browser:

1. Go to the `portfolio` folder.
2. Double‑click `index.html` (or right‑click → Open with → your browser).

## Customization checklist

Open `index.html` and search/replace:

- `Your Name`
- `Role / Domain`
- `Your City, Country`
- `Your Specialization`
- `Your Institute Name`
- All placeholder project titles, descriptions, and links.
- Contact info (email, LinkedIn, GitHub).

You can also adjust colors and spacing in `styles.css` if you want a different visual style.
